# StreamingApp

Tool for streaming anime

[Example](images/example.gif)

### Prerequisites

Tools you need to run this app

```
NodeJS
```

### Installing


```
1. Clone this repository
2. Run yarn install
```
### How to use 

Some command you can use

```
- node app.js < anime title | episode | date? >
```

exmaple

```
- node app.js vinland 16
```
